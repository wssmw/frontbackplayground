# 本地启动

## 安装依赖

```bash
pnpm install
```

## 启动后端

```bash
pnpm dev:backend
```

后端默认地址：`http://localhost:3000/api`。

Swagger 地址：`http://localhost:3000/api/docs`。

## 启动前端

```bash
pnpm dev:frontend
```

前端默认地址：`http://localhost:5173`。

## 启动文档

```bash
pnpm dev:docs
```

文档默认地址：`http://localhost:5174`。
